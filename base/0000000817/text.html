Программа recordmydesktop есть в репозитарии Ubuntu. Это консольная программа, записывает видео и звук. Захват звука можно отключить. Запись происходит в формат ogg.


Простая команда:



recordmydesktop foo.ogg



- делает запись всего рабочего стола в файл foo.ogg. Можно записывать только указанную область экрана. Формат этой команды такой:



recordmydesktop -x X_pos -y Y_pos -width WIDTH -height HEIGHT -o foo.ogg



Другие часто используемые опции:



-fps 10 - установка частоты захвата кадров в секунду


--no-sound - захватывать видео без захвата звука


--on-the-fly-encoding - кодировать видео при захвате. Если не указать, видео будет в несжатом виде сбрасваться в буфер, а после остановки записи начнется кодирование.


-v_quality 50 - качество получаемого изображения, доступные значения 0-63, по умолчанию 63


-v_bitrate 50000 - установка битрейта видеопотока, доступные значения 45000-2000000, по умолчанию 45000


--drop-frames - разрешить пропуск кадров, если не хватает времени для кодирования потока


-delay n[H|h|M|m] - задержка запуска записи в указанных единицах



Пример команды, старый формат:



recordmydesktop -x 1 -y 1 -width 1023 -height 689 -fps 10 --no-sound --on-the-fly-encoding -v_quality 47 -o video.ogg



Более новый формат:



recordmydesktop -x 1 -y 1 --width 1280 --height 720 --fps 10 --no-sound --on-the-fly-encoding --v_quality 47 -o video.ogg



Внимание! В новых версиях recordmydesktop опции наконец-то приведены к единому обозначению. Теперь все опции, которые состоят более чем из одного символа, нужно предварять двумя тире "--". Ранее только некоторые многосимвольные опции требовали двух тире.
