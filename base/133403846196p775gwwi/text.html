Можно сделать тип поля составным.


Двух вариантов - Строка и СправочникСсылка.


После чего, тип "Строка" можно будет отключить.


То есть, в типе поля в конфигураторе можно выбрать не один тип, а несколько. Для этого надо нажать кнопку с тремя точками [...].


Тогда в реквизите будет два значения храниться "Тип" и "Значение".


Старые значения будут "Строка" и "Значение строки", новые заполнишь "СправочникСсылка" и "Значение ссылки".
