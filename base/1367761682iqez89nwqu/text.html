В jQuery есть плагин поддержки протокола XML-RPC.


https://github.com/maelstrom/jquery-xmlrpc


Для получения данных с сервера используется следующий код:


alert("Before send");


// Получение XML-кода

$.xmlrpc({

url: 'JsonRpcServer',

methodName: 'getItemTable',

params: [itemId, page],

success: function(response, status, jqXHR){ alert('Success responce'); },

error: function(jqXHR, status, error) { alert('Error responce'); }

});

alert("After send");


Получение результата происходит асинхронно. То есть, последовательность появления сообщений будет следующая:


Before send

After send

Success responce
