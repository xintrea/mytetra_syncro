Иногда для захвата видео нужно указать идентификатор X-окна, с кторого будет захватываться видео. Узнать идентификатор X-окна можно командой:


xwininfo | grep "Window id" | awk '{print $4}'


После запуска этой команды появится курсор с крестиком, и им нужно кликнуть на заголовок интересуемого окна.


Результат будет примерно такой:


0x3c0000f


В программе recordmydesktop идентификатор окна указывается с помощью опции --windowid. Идентификатор указывается в полной HEX записи, то есть в начале обязательно должно быть написано "0x". Пример вызова recordmydesktop с записью видео из конкретного окна:


recordmydesktop --windowid 0x440007a --fps 10 --no-sound --on-the-fly-encoding --v_quality 47 -o video_file
