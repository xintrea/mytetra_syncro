Какие есть официальные сайты Биткоин?


  • Официальный сайт Биткоина (страница на русском языке): https://bitcoin.org/ru/


    • Какие сайты предоставляют актуальную информацию из сети Биткоин?


  • Сайт https://blockchain.info/ru/ позволяет просматривать информацию о транзакциях и адресах. Так же на этом сайте можно создать онлайновый Bitcoin-кошелек.


Как узнать курс Биткоин?


  • Самый главный сайт с курсами - http://bitcoincharts.com/
  • Самый информативный сайт с графиками курсов - https://coinmarketcap.com/
  • Весьма информативный сайт - http://bitcoinity.org/.
    На нем есть разделы Markets (текущая купля-продажа в мире) и Data (мировая статистика). Колебания цены смотрятся в разделе Markets, подраздел Price.


Где я могу найти форум пользователей системы биткоин?


  • Русскоязычную поддержку можно найти на форуме https://forum.bits.media
  • Наиболее крупным мультиязычным форумом является https://bitcointalk.org/ - там есть русская секция.


Как вводить и выводить биткоин


  • localbitcoins.com - позволяет купить биткоины за рубли через любой банк, включая Сбербанк. Так же позволяет найти пункты обмена в Росси за наличку - и для покупки, и для продажи.
  • Можно вводить - выводить через webmoney.ru. Название валюты - WMX (1 WMX равен 0,001 BTC). Покупка биткоина производится с любых вебманей, биткоины хранятся на сайте. Переброска в свой локальный или другой кошелек - через официальный сайт wmx.wmtransfer.com.
  • Схема coinbase.com -> btc-e.com -> Яндекс Деньги работает замечательно для переводов денег из любых стран в рубли. Комиссия на ввод yandex money - 3%.
  • Через онлайн-обменники, коих тысячи. Получить их список можно на сайтах https://www.bestchange.ru и https://coinmonitor.io/ru.
