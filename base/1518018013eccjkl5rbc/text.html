Для получения скриншота надо одновременно нажать две кнопки:


Кнопку питания + Кнопку уменьшения громкости


Скриншот будет сохранен в галлерее.



Для записи видео надо одновременно нажать две кнопки:


Кнопку питания + Кнопку увеличения громкости


Появится интерфейс, объясняющий, как записывать видео. Можно будет выбать качество записи. Записанное видео так же складывается в галлерею.
