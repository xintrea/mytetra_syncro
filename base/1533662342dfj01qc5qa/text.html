Если необходимо отлавливать сигналы от объекта в QML, то можно использовать 2 варианта:

1) Связывание в декларативном стиле:

Connections {

target: someObj

onSomeSignal: {

...

}



2) Связывание в императивной части (в вставках JavaScript):

someFunction() {

someObj.someSignal.connect(someFunction)

}
