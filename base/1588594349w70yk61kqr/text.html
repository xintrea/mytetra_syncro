Здесь записываются неочевидные моменты при работе с Git, которые толком не описаны в документации, или о которых постоянно забываешь.


1.


Команда git pull скачивает изменения только для текущей ветки, а не для всего репозитария. Если произошло переключение на другую ветку, для получений изменений из центрального репозитария, надо снова выполнить команду git pull.
