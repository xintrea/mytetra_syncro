В ПЭВМ Микроша имеется "пустое" пространство по адресам 8000h-BFFFh, которое по задумке разработчиков должно использоваться для подключения внешних устройств. Если ничего не подключено, в Микроше по этим адресам считывается значение 82h. Это слово состояние (status word) процессора, выдаваемое им на шину данных в начале каждого машинного цикла.
