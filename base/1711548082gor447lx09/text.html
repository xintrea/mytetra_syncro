bool myFunction(void *ptr, int x, size_t linearSize) {

(void)ptr;

(void)x;


// ... Какое-то вычисление и возврат из функции ...

if(linearSize>0)

return true;

else

return false;

}



Вопрос в том, зачем нужны вот эти строки:



(void)ptr;

(void)x;
