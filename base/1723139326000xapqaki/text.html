Программа ByeDPI для Android - это программа с открытым исходным кодом (СПО). Ее можно настраивать на различные режимы работы. Для того, чтобы восстановить работу YouTube и отключить замедление траффика, можно включить эту программу и ввести в ней специальные настройки.



Важно: Это приложение не является VPN! Оно использует VPN-режим только в пределах самого Android-устройства для переформатирования пакетов. Оно не передает ничего на удаленный VPN-сервер. Оно не шифрует трафик и не скрывает IP-адрес. ByeDPI просто создает сетевой интерфейс tun0, и на него автоматически заворачивается весь траффик Android-устройства. Далее ByeDPI берет проходящие через него пакеты и преобразовывает их согласно настройкам. По сути, ByeDPI либо работает только с заголовками, либо разбивает пакеты на более мелкие.





Установить ByeDPI можно согласно документации из официального репозитория GitHub:



https://github.com/dovecoteescapee/ByeDPIAndroid



После запуска, ByeDPI нужно оставить работать как фоновое приложение.


Само приложение состоит из одной кнопки Connect и из окна настроек. В настройки можно попасть только если соединение не установлено.


В разных регионах работают разные настройки ByeDPI. Из того, что известно и подтверждено, это то, что настройки по-умолчанию обычно не работают. Их можно заменить на примерно такие:



DNS: 8.8.8.8

Mode: VPN

Desync only HTTPS and TLS: On

Desync method: fake

Split position: 1

Split TLS record at SNI: On



После изменения настроек надо выйти из окна настроек (см. примечание ниже), и нажать кнопку Connect. Надпись на кнопке должна измениться на Connected. После этого действия можно запускать программу официального клиента YouTube.


Для справки. По-умолчанию, для версии ByeDPI 1.0.1, настройки следующие:



Theme: System

Mode: VPN

DNS: 9.9.9.9

Listen address: 127.0.0.1

Port: 1080

Maximum number of connections: 512

Buffer size: 16384

Default TTL: 0

No domain: Off

Desync only HTTPS and TLS: Off

Desync method: Disorder

Split position: 3

Split at host: Off

TTL of fake packet: 8

SNI of fake packet: www.iana.org

OOB Data: a

Host mixed case: Off

Domain mixed case: Off

Host remove spaces: Off

Split TLS record: Off

TLS record split position: 0

Split TLS record at SNI: Off


Чтобы вернуться к начальным настройкам, можно по кнопке в правом верхнем углу выбрать команду "Reset".


Примечание: В программе есть небольшой глюк: если зайти в настройки, и ничего в них не менять (хотя, даже можно и поменять), то при нажатии кнопки со стрелкой "Назад", произойдет закрытие интерфейса, а не возврат в основное окно программы. Чтобы попасть в основное окно программы, надо просто запустить ее снова через ее ярлык. Настройки будут те, которые были при последней попытке выйти из настроек.
