1. Официальная документация


Руководство программиста SH-2 (англ. SuperH SH-2 Programming Manual):

— Находится в архивах Renesas (пример).
— Описывает регистры, инструкции и тайминги.



Технические справочники для конкретных чипов (например, HD64170):

— Доступны на сайтах вроде DatasheetArchive.



2. Открытые проекты и эмуляторы


GNU Binutils (GAS, objdump):

— Поддерживает ассемблер и дизассемблирование SH-2.
— Исходный код в репозиториях GCC (GitHub).



Эмулятор MAME:

— Реализует SH-2 для аркадных автоматов (например, Sega Saturn).
— Код CPU-ядра: mame/src/devices/cpu/sh/sh2.cpp.



Проект QEMU:

— Частичная эмуляция SH-2 (менее точная, чем MAME).



3. Книги и учебные материалы


«See MIPS Run» (Dominic Sweetman):

— Сравнивает SH-2 с MIPS, объясняет RISC-принципы.



«Embedded Systems with ARM Cortex-M» (Joseph Yiu):

— Аналогии между SH-2 и ARM (полезно для понимания RISC-архитектур).



4. Онлайн-справочники


Wikibooks: SuperH:

— Краткий обзор инструкций.



CPUShack:

— Описание архитектуры.



Если нужно дизассемблировать код


Можно использовать утилиты:

— sh-elf-objdump (из пакета GNU Tools for SuperH).
— Radare2/Ghidra с поддержкой SH-2.


Например:


sh-elf-objdump -D -b binary -m sh2 firmware.bin
