В Linux имеется универсальная команда cat, которая помимо вывода содержимого файла на консоль, умеет склеивать несколько файлов. (Кстати, вывод содержимого файла в консоль, по-сути, является дублированием содержимого файла в стандартный поток вывода).


Вот команда для склейки файлов:



cat file1.txt file2.txt file3.txt > merged_file.txt



Данной команде совершенно все равно, с какими типами файлов работать. Она просто на уровне байт объединяет файлы. Поэтому такой командой можно, при необходимости, склеивать и различные бинарные файлы совершенно любого формата.
