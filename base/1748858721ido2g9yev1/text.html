Пример промпта:



Имеется следующая функция, входящая в состав PostgreSQL драйвера, используемого в фреймверке Qt. Сделай комментарий для этой функции в формате doxygen. Все параметры функции обязательно должны быть описаны в этом комментарии. Комментарий должен быть написан на русском языке, только имена параметров и прочие части синтаксиса C++ допустимо писать на английском языке.

```

Код функции

```



В первом предложении дается контекст того, что это вообще за код. Это нужно из-за того, что передается код одной-единственной функции, и сетка не видит окружающего кода. С таким промптом, на середину 2025 года, нормально работает и DeepSeek и ChatGPT.
