Пример промта:



Имеется следующий класс, входящий в состав PostgreSQL драйвера, используемого в фреймверке Qt. Добавь комментарии в код этого класса в формате doxygen. Все члены класса обязательно должны быть описаны. Комментарий должен быть написан на русском языке, только имена членов класса и прочие части синтаксиса C++ допустимо писать на английском языке.

```

код класса

```


Первое предложение необходимо для задачи контекста. Если его не прописать, генерируемые комментарии будут слишком общие.
