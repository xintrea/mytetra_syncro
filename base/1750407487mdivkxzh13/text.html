При составлении правил, которые распространяются на список хостов, перечисленных в заданном текстовом файле, надо использовать следующий принцип, который позволит обрабатывать разные опции для соответствующих списков хостов:



Нужно обязательно каждое правило закрывать условием, например –auto=none : -H list.txt -o 2 –auto=none -H list2.txt -r 2 -f -1 -t 5 –auto=none -H youtube.txt -o 2 -s -5+se –auto=none


То есть –auto=none прописано и в конце.
