В бесперебойниках китайской фирмы Kecan и дочерней компании Kehua для входа на HTTP-сервер (веб-морда) может подойти следущий вариант авторизации:



IP-адрес: 192.168.0.100

Логин: admin

Пароль: KHadmin0592



Авторизационные данные вводятся на веб-страничке, открываемой в браузере по указанному IP-адресу.
