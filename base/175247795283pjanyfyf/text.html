Большинство программистов предпочитают редактор кода Cursor с моделью нейросети Claude 3.5/3.7 Sonnet (ранее это была Claude 3 Opus). Весов данной


Для локальной работы неплохо подходит DeepSeek-Coder-V2, например DeepSeek-Coder-V2-Lite на 16 миллиардов параметров с квантизацией Q4_0. Она влезает в 24 Gb RAM и достаточно быстро работает даже без GPU-ускорителя, просто на процессоре через ollama. Ее можно удобно использовать для документирования стороннего кода в формате Doxygen. Для тех, кому приходится использовать старый сторонний код, в котором нет методичных комментариев, эта возможность очень удобна. Модель хорошо генерирует комментарии даже на русском языке, что может быть необходимо для прохождения различных сертификаций и аттестаций.
