На Youtube есть хорошее видео доклада, который называется Нарративный дизайн для соло-разработчиков (Narrative Design for Solo Devs) за авторством Nicholas O'Brien, который был проведен на конференции GodotCon 2025:



https://www.youtube.com/watch?v=f0CsbuHDr44



В нем рассказывается о том, как использовать язык ink, Open Source инструмент inky и объект InkPlayer в движке Godot для создания квестов и прочих историй с выбором действия, который совершает персонаж.
