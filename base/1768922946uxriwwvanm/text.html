Чтобы проверить как работает VLESS и сделать первые шаги по разблокированию интернет-ресурсов, можно воспользоваться двумя следующими VLESS-приложениями.



V2RAYNG


Частичное описание работы находится здесь: https://github.com/2dust/v2rayNG/wiki/Mode


На GitHub регулярно выкладываются релизы. Не стоит бояться иероглифов в описании - разработчики являются китайцами, и это ПО используется для обхода Великого китайского фаирвола. Так же релизы есть в Google PLay и в F-Droid.


Ссылка на инициализирующие данные:



https://raw.githubusercontent.com/roosterkid/openproxylist/main/V2RAY_RAW.txt

https://github.com/AvenCores/goida-vpn-configs/raw/refs/heads/main/githubmirror/1.txt

https://github.com/AvenCores/goida-vpn-configs/raw/refs/heads/main/githubmirror/2.txt

https://github.com/AvenCores/goida-vpn-configs/raw/refs/heads/main/githubmirror/3.txt



Если с ними не заработает, конфиги можно найти здесь:



https://github.com/AvenCores/goida-vpn-configs



Там же есть описание на русском языке как конфигурировать.


В некоторых версиях имеется небольшая проблема: если добавить более чем 3 ссылки из буфера обмена, будет ошибка. Но обычно достаточно 3 ссылок.
