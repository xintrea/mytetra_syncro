Монитор работает, но иногда мигает - то есть, пропадает изображение на секунду, виден черный экран. Слышан тихий высокочастотный писк.


Плата блока питания, маркировка: ILPI-170 REV: B 2009.11.26


Плата однослойная, односторонняя: с одной стороны дорожки, с другой электронные элементы и перемычки.



Высоковольтная область


В центре платы выделена толстой линией высоковольтная область, на которой распаян разъем 220В.


На другую часть платы, из этой области есть выход только через крупный трансформатор SPW-132 RoHS NCE N 0942 A GH-130 E217615.


Так же на другую часть платы из высоковольтной области торчит оптопара L0946 817B X.


В этой высоковольтной области есть 8-ножечный шим-контроллер TKSAH P0368 MYA. Говорят, что для него есть аналог LD7575 (или LD7575PS, информация отсюда: https://forum-monitor.net.ru/threads/392030/).


С одной стороны платы есть гребенка для питания материнской платы формирования видеосигнала. На той же стороне сделан усилитель звукового сигнала, который поступает со звуковых разъемов. На этой гребенке следующие контакты:


1 -2. USB 5V

3/5. VCC 5V
